### Adjacency Matrix Graph

- Adjacency Matrix를 이용한 data struct Graph입니다.
- 코드는 "input1.txt"를 입력받아서 이것을 DFT와 BFT의 출력을 보이고 있습니다.
- Graph는 특별한 조건없이 vertex가 arc로 연결 되어있으므로 입력을 받아서 존재하지 않는 벌텍스면 새로이 제작하여 arc로 연결했습니다.
- 또한 아크들은 2X2 matrix를 이용하여 제작하였습니다.
- matrix의 1은 연결을 0은 연결되지 않음을 의미합니다.
- BFT는 Queue의 특징을 가지고 있기 때문에 Queue를 이용하여 제작하였습니다.
