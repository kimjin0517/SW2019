### Adjacency Weight List Graph

- Adjacency Weight List를 이용한 data struct Graph입니다.
- 코드는 "input2.txt"를 입력받아서 Shortest Path와 Minimmun Spanning Tree를 출력으로 보이고 있습니다.
- Adjacency Weight List Graph는 Adjacency List의 가중치를 포함시켜 제작하였습니다.
- Minimmun Spanning Tree는 prim jarnik algorithm을 이용하여 제작하였습니다.
- https://ko.wikipedia.org/wiki/%ED%94%84%EB%A6%BC_%EC%95%8C%EA%B3%A0%EB%A6%AC%EC%A6%98
- Shortest path는 Dijkstra's algorithm을 이용하여 제작하였습니다.
- https://en.wikipedia.org/wiki/Dijkstra%27s_algorithm
